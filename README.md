## 🔧 快速开始

### 环境要求
- 现代浏览器 (Chrome 60+, Firefox 55+, Safari 12+)
- n8n 服务器环境
- 硅基流动 API 访问权限

### 使用说明

1. 在"图像描述提示词"中输入您想要生成的图像描述
2. 调整图像宽度和高度（可选）
3. 设置随机种子（可选，用于重现相同结果）
4. 点击"开始创作图像"按钮
5. 等待FOXAI AI处理并生成图像
6. 查看结果并下载您的创作

### 部署方式

1. **直接使用**: 在浏览器中打开 `flux-image-generator.html`
2. **Web服务器**: 部署到Web服务器提供在线服务
3. **本地开发**: 配置本地开发环境
